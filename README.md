## Pizza-Sales-Analytics-End-to-End

# Project Overview
- This project analyzes pizza sales data using SQL and Power BI to uncover business insights related to revenue, orders, sales trends,      product performance, and customer purchasing behavior.
- The objective is to transform raw sales data into actionable insights through data analysis and interactive dashboard visualization.

# Dashboard Preview
# Key Insights
# Busiest Days
* Highest orders are generated during weekends.
* Friday and Saturday contribute the highest number of sales.
# Peak Months
* July and January show the highest sales volume.
# Best Performing Category
* Classic Pizza category contributes the highest revenue and total orders.
# Size Performance
* Large-sized pizzas account for the highest percentage of sales.
